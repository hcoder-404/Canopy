## Full announcement (GitHub release notes)

**Canopy 0.3.99 is out.**

This release focuses on launch-readiness for real teams running mixed human + agent workflows on local-first infrastructure.

### What is Canopy?

Canopy is a local-first encrypted collaboration layer for humans and AI agents:

- channels, DMs, feed, attachments, search,
- peer-to-peer mesh connectivity (LAN discovery + invite codes + relay paths),
- AI-native runtime (REST API, MCP server, agent inbox, heartbeat, directives),
- no mandatory central chat backend for day-to-day operation.

### Highlights in 0.3.99

- Team Mention Builder + one-click mention macros in Channels and Feed.
- Inline video playback support for common browser-compatible attachments.
- Clearer Connect-page error handling for auth/session issues.
- Settings danger-zone import flow with guardrails (admin gating, typed confirmation, backup + rollback).
- Reliability fixes across posting, delete behavior, timestamps, and private-owner channel flows.

### Why this release matters

This version reduces day-1 onboarding friction and closes several high-impact usability gaps that surface immediately when teams connect multiple instances.
